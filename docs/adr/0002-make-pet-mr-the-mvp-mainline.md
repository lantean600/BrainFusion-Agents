# Make PET/MR the MVP mainline

Status: accepted

The publishable MVP is a brain PET/MR multi-agent fusion workflow using ADNI as the primary dataset and OASIS-3 as external validation. This keeps the core experiment aligned with paired brain multimodal imaging while letting the agent system demonstrate QC, failure recovery, evidence tracking, and workflow robustness around conventional fusion models.

## Consequences

- PET-only, MR-only, and PET/MR fusion baselines are mandatory MVP comparisons.
- WSI and CT work should not block PET/MR progress.
- The multi-agent contribution must be evaluated as workflow robustness, not as a replacement for fusion modeling.
